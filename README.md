# Athlete Performance, Recovery & Injury Risk Analysis

## Overview
This project analyzes 1,000 athlete records to explore how sleep, recovery, fatigue, training intensity, and sport type relate to performance and injury risk.

## Objectives
- Compare average performance across sports
- Examine recovery differences across sports
- Analyze fatigue trends by training intensity
- Compare performance across sleep groups
- Explore differences between performance groups and readiness metrics

## Conclusion
This project highlights how athlete performance is influenced by multiple factors beyond a single readiness metric. The findings suggest that stronger performance can coexist with elevated fatigue and lower recovery, reinforcing the importance of monitoring multiple indicators together

## Tools Used
- MySQL
- MySQL Workbench
- Tableau
- CSV

## Dataset
The dataset contains 1,000 athlete records with variables including sport type, training intensity, sleep hours, fatigue level, recovery index, performance score, and injury risk.

## Process
1. Imported the CSV into MySQL Workbench
2. Cleaned and standardized columns
3. Queried the dataset using SQL
4. Built a one-page Tableau dashboard
5. Summarized findings with visual insights

## Key Insights
- Tennis and Football led the dataset in average performance
- Athletics had the highest average recovery index
- Athletes with 8+ hours of sleep recorded the highest average performance
- Higher-performing athletes also showed higher fatigue and lower recovery
